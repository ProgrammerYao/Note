# 重要内容

有什么(理论内容)、做什么

==导论==

推进马克思主义中国化现代化的必然性：

* (1)理论本身发展的内在要求
* (2)解决中国实际问题的客观需要

准确把握马克思主义中国化时代化的科学内涵，要做到：

* (1)坚持马克思主义与发展马克思主义相统一。
* (2)坚持和发展马克思主义，必须同中国具体实际相结合、同中华优秀传统文化相结合。

==毛泽东思想==

三大作风：理论联系实际、密切联系群众、批评与自我批评的作风（“厘米屁”）

“两个务必”：

* (1)务必使同志们继续地保持谦虚、谨慎、不骄、不躁的作风
* (2)务必使同志们继续地保持艰苦奋斗的作风。

==新革==

俄国十月革命：标志中国资产阶级民主革命的时代背景发生了根本转换，标志着人类历史开始了由资本主义向社会主义转变

五四运动：标志近代中国进入新革阶段

革命任务两步走：

* 第一步是完成反帝反封建的新民主主义革命任务
* 第二步是完成社会主义革命任务

新革总路线内容：无产阶级领导的，人民大众的，反对帝国主义、封建主义和官僚资本主义的革命

新革的对象、动力、领导力量、性质和前途

* 对象：帝国主义、封建主义和官僚资本主义
* 动力：无产阶级、农民阶级、城市小资产阶级和民族资产阶级
* 领导力量：无产阶级
* 性质：资产阶级民主主义革命
* 前途：社会主义

区分新旧民主主义革命的根本标志：革命的领导权在无产阶级还是资产阶级

新革纲领：

* 政治纲领是：推翻帝国主义和封建主义的统治，建立一个无产阶级领导的、以工农联盟为基础的、各革命阶级联合专政的新民主主义共和国
* 经济纲领是：没收封建地主阶级的土地归农民所有，没收官僚资产阶级的垄断资本归新民主主义的国家所有，保护民族工商业
* 文化纲领是：无产阶级领导的人民大众的反帝反封建的文化，即民族的科学的大众的文化。

新革道路：农包城、武夺政的革命道路

新革道路形成的必然性：中国所处的时代特点和具体国情

新革道路的根本内容：处理好土地革命、武装斗争、农村革命根据地建设(“土豆村”)关系

新革的三大法宝：统一战线、武装斗争、党的建设

革命的联盟：工农联盟、与民资的联盟

==社革(社改)==

新民主义主义社会是中华人民共和国成立到社会主义改造基本完成的过渡时期，性质是新民主主义社会，但不是一个单独的社会形态

新民主主义社会的五种经济成分（“个国两资集体合作”）：

```tx

| 经济成分          | 对应的阶级/过渡形式                 | 性质       |
| :-------------: | :--------------------------: | :--------: |
| **社会主义性质**的国营经济   | 工人阶级                       | 三种主要经济成分 |
| 农民和手工业者的个体经济  | 农民阶级和其他小资产阶级               |    ^^      |
| 私人资本主义经济      | 民族资产阶级                     |     ^^     |
| **半社会主义性质**的合作社经济 | **个体经济**向**社会主义<br>集体经济**过渡的形式     | 过渡性的经济   |
| 国家资本主义经济      | **私人资本主义经济**向<br>**社会主义国营经济**过渡的形式 |     ^^     |
```

随着土地改革的完成，工人阶级和资产阶级的矛盾成为我国社会的主要矛盾

过渡时期总路线：一化三改

* 一化：社会主义工业化
* 三改：对个体农业、手工业、资本主义工商业进行社会主义改造

社改的中心任务是社会主义工业化建设

社会主义革命的目的是为了解放生产力

三改阶段和方式：
```tx

| 总结：农业、手工业、资本主义工商业社会主义改造的阶段 ||||
| :--------------------------: | :--------------: | :------------: | :-----------: |
|                            | 农业             | 手工业          | 资本主义工商业     |
| 社会主义萌芽性质                     | 互助组            | 供销小组         | 初级形式的国家资本主义 |
| 半社会主义性质                      | 初级社            | 供销合作社        | 个别企业的公私合营   |
| 完全社会主义性质                    | 高级社            | 生产合作社        | 全行业的公私合营    |
|                  <br>           ||||
| 过渡形式                       | 合作化            || 国家资本主义      |
| 方法                         | 引导、说服和教育       | 说服教育、示范和国家帮助 | 和平赎买        |
| 都采取的方式                     | 区别对象、积极引导、逐步过渡 |||

```

社会主义改造的基本完成：标志着中国历史上长达数千年的阶级剥削制度的结束和社会主义基本制度的确立，标志着我国由一个新民主主义的国家转变为社会主义国家，标志着我国进入了社会主义初级阶段。

第一届全国人民代表大会的召开和《中华人民共和国宪法》的制定及颁布施行：成为我国社会主义民主政治建设的里程碑。这部宪法明确规定了我国人民民主专政的国体和人民代表大会制度的政体。

社会主义政治制度体系（“人多自愈“）：

* 人民代表大会制度(根本政治制度)
* 中国共产党领导的多党合作和政治协商制度(基本政治制度)
* 民族区域自治制度(基本政治制度)
* 基层群众自治制度(基本政治制度)

==社建==

初步探索的理论成果主要包括：调动一切积极因素为社会主义事业服务的思想；正确认识和处理社会主义社会矛盾的思想；走中国工业化道路的思想（3思）

如何调动一切积极因素为社会主义事业服务：坚持中国共产党的领导，发展社会主义民主政治

《论十大关系》的第一大关系是重工业和轻工业、农业的关系

《论十大关系》标志着党探索中国社会主义建设道路的良好开端

社会主义的基本矛盾：生产关系和生产力之间的矛盾，上层建筑和经济基础之间的矛盾（一直不变）

经济建设的主要任务是实现国家工业化

如何走中国工业化道路：

* (1)必须采取正确的经济建设方针。
* (2)必须调整和完善所有制结构。
* (3)必须积极探索适合我国情况的经济体制和运行机制。

工业化发展思想：以工业为主导，重工业作为重点，农轻重为序，以及“两条腿走路”的工业化发展思路，即重工业和轻工业同时并举，中央工业和地方工业同时并举，沿海工业和内地工业同时并举，大型企业和中小型企业同时并举，等等

人物成果：

* 毛泽东、刘少奇、周恩来提出了把资本主义经济作为社会主义经济的补充的思想。
* 毛泽东提出“两参一改三结合”。
* 朱德提出了要注意发展手工业和农业多种经营的思想。
* 陈云提出了“三个主体，三个补充”的设想。
* 邓小平提出实行职工代表大会制等思想。（“邓职”）

其它成果：

* 社会主义发展阶段分为两个：不发达的社会主义、比较发达的社会主义（后比前时间更长）
* 四个现代化战略目标：周恩来提为农业、工业、国防、科学技术 现代化
	* 科学技术：党提出“向科学进军口号” ，强调科学技术现代化是四个现代化的关键
	* 国防：“三五”计划被概括为“备战、备荒、为人民”。
* 四个现代化战略步骤：第一步要建立一个独立的比较完整的工业体系和国民经济体系，第二步要全面实现“四个现代化”使中国的经济发展走在世界的前列。
* 祖国统一：周恩来提“一纲四目”解决台湾问题
