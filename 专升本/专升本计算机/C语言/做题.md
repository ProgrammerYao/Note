# 程序运行题

如果是分支结构，则画出每个分支控制的范围，部分操作可直接旁边写出

如果是循环结构，则使用`数字1、数字2...数字n×`在循环旁边标注判断值，部分操作可直接旁边写出

打印浮点数时记得结果小数部分是保留六位

# 手写程序题

模板：
```C
# include<>

void main()
{
	
}
```

`printf()`函数每次输出最后添加换行符`\n`

数组操作时，使用数组下标遍历进行操作比较好

==书写规范==

1.没有≤，只有＜＝

2.一个if是一个语句，但if...else是两个语句要{}

3.字符的判断要用''扩起

4.书写条件、循环、函数时直接使用`{}`，防止写漏

5.字符的输入和输出采用`getchar()`和`putchar(字符)`即可，字符串的输入和输出采用`gets(地址)`和`puts(地址)`即可

6.判断字符串是否越界：`arr[i++]!='\0'`

==注意事项==

自然数从0开始
